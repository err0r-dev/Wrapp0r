# Audio Tracks

Place royalty-free audio files in this directory. The expected files are:

- `energetic.mp3` - High-energy electronic/EDM for fitness data
- `chill.mp3` - Relaxed lo-fi beats for productivity data
- `upbeat.mp3` - Positive pop/electronic for music data
- `dramatic.mp3` - Cinematic ambient for dramatic reveals
- `warm.mp3` - Acoustic/jazz for food data
- `professional.mp3` - Corporate ambient for finance data

## Recommended Sources

All tracks should be royalty-free for commercial use:

- [Pixabay Music](https://pixabay.com/music/) - Free, no attribution required
- [Free Music Archive](https://freemusicarchive.org/) - Various CC licenses
- [Uppbeat](https://uppbeat.io/) - Free tier available
- [Mixkit](https://mixkit.co/free-stock-music/) - Free, no attribution required

## Audio Specifications

For best results:
- Format: MP3
- Bitrate: 128-192 kbps
- Duration: 2-4 minutes (will loop)
- Style: Instrumental without vocals
